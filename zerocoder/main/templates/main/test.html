Взаимодействие {{caption}} с PostgreSQL (дополнение к уроку №2)

Основные шаги настройки:

  1. Установите драйвер: pip install psycopg2
  2. В settings.py настройте базу данных:

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'mydatabase',
        'USER': 'myuser',
        'PASSWORD': 'mypassword',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
  1. Примените миграции: python manage.py migrate
  2. Создайте суперпользователя: python manage.py createsuperuser
  3. Запустите сервер: python manage.py runserver
